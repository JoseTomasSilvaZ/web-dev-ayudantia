Arrays

Los arrays son listas de elementos, los cuales pueden ser de cualquier tipo. 

            
              // Declare an array in JS
              const myFavoriteAnimals = ["🐶", "🐱", "🐭", "🐹", "🐰"];

              // Access to element of array
              // In this example, we are accessing to the first element of the array and printing it in the console
                console.log(myFavoriteAnimals[0]); //🐶

              // Add an element to the array
                myFavoriteAnimals.push("🐻"); 
                console.log(myFavoriteAnimals); // ["🐶", "🐱", "🐭", "🐹", "🐰", "🐻"]

              // Remove last element of the array
                myFavoriteAnimals.pop();
                console.log(myFavoriteAnimals); // ["🐶", "🐱", "🐭", "🐹", "🐰"]

              // Remove first element of the array
                myFavoriteAnimals.shift();
                console.log(myFavoriteAnimals); // ["🐱", "🐭", "🐹", "🐰"]

              // Add an element to the beginning of the array
                myFavoriteAnimals.unshift("🐶");
                console.log(myFavoriteAnimals); // ["🐶", "🐱", "🐭", "🐹", "🐰"]

              // Find the index of an element
                console.log(myFavoriteAnimals.indexOf("🐭")); // 2

Los arrays tienen muuuchos métodos que son de gran ayuda.

Objetos

Los objetos son una colección de datos y/o funcionalidades.
Estos datos y funcionalidades son llamados propiedades y métodos, respectivamente.
Las propiedades son pares clave-valor, donde la clave es un string (también llamado nombre de la propiedad) y el valor puede ser cualquier cosa.
Los métodos son funciones que pueden ser llamadas en un objeto. 

                
                    // Declare an object in JS
                    const myFavoriteAnimal = {
                        name: "🐶",
                        age: 3,
                        isCute: true,
                        bark: function() {
                            console.log("Woof woof");
                        }
                    };

                    // Access to property of object
                    console.log(myFavoriteAnimal.name); // 🐶

                    // Access to method of object
                    myFavoriteAnimal.bark(); // Woof woof

                    // Add a new property to the object
                    myFavoriteAnimal.color = "brown";
                    console.log(myFavoriteAnimal); // {name: "🐶", age: 3, isCute: true, bark: ƒ, color: "brown"}
